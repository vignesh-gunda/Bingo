# Called Numbers Display (Mobile Screen View)

## Feature Description
This document describes how the numbers called during a Bingo game are displayed to the user on a mobile device, specifically showing the most recently called number and only the previous number.

## Visual Elements
*   **Current Called Number:**
    *   Prominent, large display of the MOST recently called Bingo number.
*   **History of Called Numbers:**
    *   A list or scrolling display of the last number called in the current round. Only the last number and the current number is shown

## User Interaction
*   Users can quickly identify the current number.
*   Interface should be clear and not interfere with card interaction.
*   Called numbers should be in a single horizontal line above the user's numbers 

## Mobile Considerations
*   Current number display must be highly visible even if the user is focused on their card.
